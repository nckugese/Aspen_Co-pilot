# HyCyc — Advanced

See [common-advanced.md](common-advanced.md) for property method override, electrolyte, free-water, EO, utility, and CO2e tracking paths shared by all blocks.

The HyCyc block has extensive geometry ratio and efficiency correlation parameters (Plitt, Nageswararao) configured through sub-nodes. Note that HyCyc does not have utility assignment or utility cost outputs.
