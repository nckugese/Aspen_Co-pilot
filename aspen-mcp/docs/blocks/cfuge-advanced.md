# Cfuge — Advanced

See [common-advanced.md](common-advanced.md) for property method override, electrolyte, free-water, EO, utility, and CO2e tracking paths shared by all blocks.

The Cfuge block has extensive sub-models for classification, deliquoring, and cake properties. These are configured through the block's sub-nodes depending on the selected centrifuge type. Note that Cfuge does not have utility assignment or utility cost outputs.
