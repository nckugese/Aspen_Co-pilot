# Fluidbed — Advanced

See [common-advanced.md](common-advanced.md) for property method override, electrolyte, free-water, EO, utility, and CO2e tracking paths shared by all blocks.

The Fluidbed block has extensive parameters for distributor design, heat exchanger tubes, elutriation models, and TDH correlations configured through sub-nodes. Note that Fluidbed does not have utility assignment or utility cost outputs.
